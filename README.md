# CLI Hex Analyzer & Dumper v1.0.0

Легковесная консольная утилита на языке C, предназначенная для низкоуровневого анализа структуры бинарных файлов, поиска сигнатур (магических байт) и реверс-инжиниринга.

## 🚀 Как запустить (Для пользователей)
Вы можете не компилировать код самостоятельно, а скачать уже готовую собранную версию программы для Windows:
👉 [Скачать готовый HEX-dumper.exe](https://github.com/king77743/HEX-dumper/releases/download/v1.0/HEX-dumper.exe)

1. Скачайте файл по ссылке выше.
2. Запустите его двойным кликом мыши или через терминал.
3. Введите путь к любому файлу (например, C:\папка\file.jpg) и нажмите Enter.

---

## 🛠 Особенности и возможности программы
* Цветная подсветка терминала: Адреса смещения (offset) выделены голубым цветом, а ASCII-инспекция — зелёным. Это позволяет быстро сканировать структуру файла глазами.
* Фильтрация бинарного мусора: Все нечитаемые и служебные байты автоматически заменяются точкой ., что защищает кодировку вашей консоли от крашей и ломаных символов.
* Идеальная геометрия колонок: Специальный алгоритм EOF-паддинга динамически рассчитывает недостающее смещение в конце файла и добивает строку пробелами. Колонки всегда остаются идеально ровными.
* Минимальное потребление RAM: Программа использует кольцевой 16-байтовый буфер и потоковое чтение fgetc. Она способна обрабатывать файлы объёмом в десятки гигабайт без нагрузки на систему.

## 💻 Сборка из исходного кода (Для разработчиков)
Если вы хотите скомпилировать проект самостоятельно, вам понадобится компилятор GCC (стандарт C99/C11):
```
gcc work_file.c -o HEX-dumper.exe
```
